## Where to put styles

Embedded, external, inline. So far, we've seen embedded.

With both embedded and external styles, all rules are **global**!

### External styles

Can be reused between HTML files.

### Inline styles

Useful for prototyping and for passing variables to CSS. Hard to maintain.

## Selectors

The 3 most common: Tag, id, and class.

<Example
  title="Tag, id, and class selectors"
  preset="html"
  height={520}
  files={{
    'index.html': `<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <title>My page</title>
    <style>
      p {
        color: red;
      }
      .my-paragraph {
        color: blue;
      }
      #my-example {
        color: green;
      }
    </style>
  </head>
  <body>
    <p id="my-example" class="my-paragraph">Welcome!</p>
  </body>
</html>`,
  }}
/>
