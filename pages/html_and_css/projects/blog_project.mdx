## Steps

1. Create or open an HTML file. Add a `<DOCTYPE>`, `<html>`, `<head>`, and `<body>`.

1. Add a `<style>` tag. Add a rule to style the `body` element - use `rgb(243, 238, 255)` for the background color and `rgb(26, 1, 114)` for the text color.

1. Add a `<header>`, `<main>`, and `<footer>`. Each of these should have `120px` horizontal padding and `40px` vertical padding. The `header` and `footer` should use the background color `aliceblue`.

1. Finish the `header`: add an `<h1>` element

1. Finish the `footer`: add a copyright - use an HTML entity for the copyright symbol.

1. Now for the `main` section:

   - Create two `<section>` elements, each with an `<h2>` and some paragraphs (`<p>`). Sections should have a white background and `40px` padding. They should be separated from each other by `40px`.

   - Use `<strong>` and `<em>` to bold and italic some text.

   - Use `<a>` to create a link. You may use `https://picsum.photos/` for a random link, or something else of your choice.

   - Use `<img>` to create an image. You may use `https://picsum.photos/300/200` for a random image, or something else of your choice.

> Tip: You may need to reset some of the browser's built-in styles for elements to acheive the right margin and padding. For example, the `body` element typically has a built-in margin.
