## What is responsive design?

Responsive design means making a website adapt to different devices and screen sizes.

By default, many HTML elements automatically resize to fill the width of their parent element, so an HTML page should still be _usable_ regardless of screen size. However, this can sometimes result in awkwardly large or small elements on some screen sizes.

## Media queries

Media queries are a tool for applying different CSS rules to different screen sizes.

For example, suppose we wanted to use a different image size depending on the size of the screen:

<Example
  title="CSS Styles"
  preset="html"
  height={500}
  files={{
    'index.html': `<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <title>My page</title>
    <style>
      body {
        text-align: center;
      }
      img {
        width: 400px;
        height: 300px;
      }
      @media (max-width: 600px) {
        img {
          width: 200px;
          height: 150px;
        }
      }
    </style>
  </head>
  <body>
    <img alt="dog" src="https://picsum.photos/id/237/200/150">
  </body>
</html>`,
  }}
/>

Here's an example where we use a media query to change the root font size - and since all headings have rem-based font sizes by default, they all resize!

<Example
  title="CSS Styles"
  preset="html"
  height={500}
  files={{
    'index.html': `<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <title>My page</title>
    <style>
      html {
        font-size: 150%;
      }
      @media (max-width: 1200px) {
        html {
          font-size: 112.5%;
        }
      }
      @media (max-width: 600px) {
        html {
          font-size: 87.5%;
        }
      }
    </style>
  </head>
  <body>
    <h1>Heading 1</h1>
    <h2>Heading 2</h2>
    <h3>Heading 3</h3>
  </body>
</html>`,
  }}
/>
