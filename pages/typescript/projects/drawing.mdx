2. Implement the pencil tool.

3. Add one or more of the following features:

   - **Keyboard Shortcuts**

     Pressing "r" should activate the rectangle tool, "o" should activate ellipse, and "p" or "v" should activate pencil.

     > Hint: use the `onkeypress` event, and set the `checked` property of the input elements

   - **Eraser Tool**

     When the eraser tool is active, dragging should remove any shape under the mouse.

     > Hint: use the canvas APIs `context.isPointInPath` and `context.isPointInStroke` to determine the shape under the mouse

   - **Move Tool (hardest)**

     When the move tool is active, dragging should move the top shape under the mouse.

     > Hint: use the canvas APIs `context.isPointInPath` and `context.isPointInStroke` to determine the shape under the mouse
